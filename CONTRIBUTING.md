# 贡献指南

感谢您对Single-Tree Segmentation项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果您发现了bug或有功能建议，请通过GitHub Issues提交。在提交前，请先搜索是否已有相关问题。

### 提交代码

1. Fork本项目
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个Pull Request

### 代码规范

- 遵循现有的代码风格
- 添加适当的注释
- 确保代码可以编译通过
- 如果可能，添加测试用例

## 开发环境设置

1. 克隆仓库
```bash
git clone https://github.com/yourusername/Single-tree.git
cd Single-tree
```

2. 编译项目
- Windows: 使用Visual Studio打开 `Single-tree.sln`
- Linux/macOS: `g++ -std=c++11 -O2 Single-tree/Single-tree.cpp -o single_tree_segmentation`

## 许可证

通过贡献代码，您同意您的贡献将在MIT许可证下发布。
